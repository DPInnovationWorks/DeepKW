# DeepKW 1.0.0

**project structure**

    ```
    project
    │   package.json    
    │   package-lock.json
    │   webpack.config.js
    │
    └───src
    │   │   index.html
    │   │   index.js //入口文件
    │   │   app.vue
    │   │
    │   └───css
    │   │
    │   └───components //组件部分
    │   │
    │   └───router
    │   │   router.js 
    │   
    └───dist //生成
        │   index.html
        │   bundle.js
    ```


**Project Operation**
The following operations will instantly start the project:
```
cd project

npm install

npm run dev

```
